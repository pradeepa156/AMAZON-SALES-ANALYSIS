## AMAZON SALES DATA ANALYSIS AND INTERACTIVE DASHBOARD

---

# 1. ABSTRACT

This project presents the development of an interactive **Amazon Sales Dashboard** designed to analyze and visualize sales performance across product categories, customer regions, and time periods. The dashboard was created using sales data containing product categories, prices, revenues, ratings, customer regions, and order dates.

The main objective of this project is to transform raw sales data into meaningful business insights using data visualization techniques. The dashboard provides dynamic filtering options and graphical representations that assist in understanding total sales, revenue trends, regional performance, product category distribution, and customer satisfaction ratings.

This system enables decision-makers to identify high-performing products, profitable regions, and overall sales trends efficiently. The project demonstrates practical implementation of Business Intelligence concepts using data analytics and visualization tools.

# 2. INTRODUCTION

In today’s digital economy, e-commerce platforms generate vast amounts of transactional data. Analyzing this data is essential for understanding customer behavior, improving sales strategies, and maximizing revenue.

Amazon, being one of the largest online retailers, operates across multiple product categories and regions. Effective sales analysis can help businesses:

* Identify best-selling product categories
* Monitor revenue performance
* Track customer ratings
* Analyze regional sales contribution
* Evaluate discounted pricing impact

This project focuses on building a **Sales Dashboard** that provides an interactive and visual representation of Amazon sales data.

# 7. SYSTEM METHODOLOGY

The project was completed in the following stages:

### 7.1 Data Collection

The dataset was obtained in CSV format containing Amazon sales transactions.

### 7.2 Data Cleaning

* Removed blank records
* Checked for missing values
* Verified data types
* Corrected formatting issues

### 7.3 Data Processing

* Created calculated fields (Total Sales, Revenue)
* Aggregated data using Pivot Tables
* Generated summary metrics

### 7.4 Dashboard Design

The dashboard includes:

* KPI Cards (Total Sales, Total Rating, Min Sales, Max Sales)
* Bar Charts
* Pie Charts
* Donut Charts
* 3D Column Charts
* Slicers for dynamic filtering
